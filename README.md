# ChronoGlobe ULTRA

![ChronoGlobe ULTRA Banner](https://picsum.photos/seed/chronoglobe/1200/400?blur=2)

**ChronoGlobe ULTRA** is a premium, high-performance world clock application designed for modern workflows. Built with React, TypeScript, and Tailwind CSS, it features a stunning glassmorphism UI, multiple clock faces, and intelligent timezone management.

## 🌟 Features

- **Multiple Clock Faces**:
  - **Glyph**: A futuristic, decoding-style digital display.
  - **Classic Digital**: High-contrast, legible design for professional environments.
  - **Neon**: Cyberpunk-inspired aesthetic with glowing effects.
  - **Minimal**: Clean, distraction-free interface.
- **Intelligent Timezone Management**:
  - Real-time conversion across major global business hubs.
  - Visual day/night indicators.
  - Sunrise and sunset tracking.
- **Smart Theming**:
  - Automatic dark/light mode switching based on system preference.
  - Adaptive gradients and shadows for optimal readability in any lighting.
- **Productivity Tools**:
  - **Global Date Tracker**: Visualize day of year and week number.
  - **Meeting Planner**: Quickly check overlapping business hours.
  - **Reminders**: Set local alerts for specific timezones.

## 🚀 Tech Stack

- **Frontend Framework**: [React 18](https://reactjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/) (formerly Framer Motion)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 🛠️ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/chronoglobe-ultra.git
    cd chronoglobe-ultra
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```

## 📂 Project Structure

```
/src
  /components       # Reusable UI components (ClockFaces, Panels, etc.)
  /hooks            # Custom React hooks (useWorldTime, etc.)
  /assets           # Static assets
  App.tsx           # Main application entry
  main.tsx          # React DOM rendering
  index.css         # Global styles & Tailwind imports
```
